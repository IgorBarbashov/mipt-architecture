1. Создаем сервис аутентификации, который проверяет логин и пароль пользователя, создает и возвращает JWT в куки. В JWT должен быть "зашит" id пользователя. Работу с базой реализовывать не надо!

2. Создаем сервис сообщений, принимающий текст сообщения от пользователя с JWT в куки. Он должен проверять срок действия и валидность JWT и отвечать 201 или 401.
